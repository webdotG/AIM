
Система для создания записей с поддержкой черновиков, автосохранения и переиспользуемых компонентов.

1. Разделение ответственности по сторам
API Stores (работают только с API)
EntriesStore - CRUD операций с записями

EmotionsStore - работа с эмоциями и справочником

BodyStatesStore - состояния тела

CircumstancesStore - обстоятельства

SkillsStore - навыки

TagsStore - теги

RelationsStore - связи между записями

SkillProgressStore - прогресс навыков

Draft Store (только черновик формы)
EntryDraftStore - хранение черновика новой записи, автосохранение в LocalStorage

2. Гибридные компоненты-пикеры
Каждый пикер поддерживает два режима работы:

Режим 1: Автономный (для переиспользования)
javascript
<EmotionPicker
  selectedEmotions={selectedEmotions}  // данные из родителя
  onChange={handleEmotionsChange}      // колбэк для обновления
  maxEmotions={5}
/>
Режим 2: Интегрированный с черновиком (для EntryForm)
javascript
<EmotionPicker
  draftStore={entryDraftStore}    // стор черновика
  draftField="emotions"           // поле в черновике
  maxEmotions={5}
/>

Пример EmotionPicker:

javascript
const EmotionPicker = ({
  // Автономный режим
  selectedEmotions,
  onChange,
  
  // Интегрированный режим
  draftStore,
  draftField = 'emotions',
  
  // Общие
  maxEmotions = 5,
  onClose
}) => {
  // Определяем режим
  const isDraftMode = !!draftStore && !!draftField;
  
  // Получаем данные
  const currentSelection = isDraftMode 
    ? draftStore.currentDraft[draftField]
    : selectedEmotions;
  
  // Обработчик изменений
  const handleChange = (newItems) => {
    if (isDraftMode) {
      draftStore.updateDraft({ [draftField]: newItems });
    } else {
      onChange?.(newItems);
    }
  };
  
  // Рендер UI...
};
EntryForm (главная форма)
Импорты:

javascript
import { 
  useEntryDraftStore,  // только черновик
  useEntriesStore,     // для сохранения через API
  useUIStore          // для уведомлений/ошибок
} from '@/store/StoreContext';
Структура:

javascript
const EntryForm = () => {
  // Сторы
  const entryDraftStore = useEntryDraftStore();  // черновик
  const entriesStore = useEntriesStore();        // API для записей
  
  // Состояние
  const [isSubmitting, setIsSubmitting] = useState(false);
  
  // Данные формы
  const draft = entryDraftStore.currentDraft;
  
  // Обработчик отправки
  const handleSubmit = async () => {
    // Валидация
    if (!draft.content.trim()) return;
    
    setIsSubmitting(true);
    try {
      // Сохраняем через API
      await entriesStore.createEntryFromDraft(draft);
      
      // Очищаем форму
      entryDraftStore.resetDraft();
      
      // Уведомление об успехе
    } catch (error) {
      // Обработка ошибок
    } finally {
      setIsSubmitting(false);
    }
  };
  
  return (
    <form onSubmit={handleSubmit}>
      {/* Все пикеры в интегрированном режиме */}
      <EmotionPicker
        draftStore={entryDraftStore}
        draftField="emotions"
      />
      
      <TagsPicker
        draftStore={entryDraftStore}
        draftField="tags"
      />
      
      {/* ... другие пикеры */}
      
      <button type="submit">
        Создать запись
      </button>
    </form>
  );
};
